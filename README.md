# 🚀 AI Career Guidance System

An intelligent web application that suggests career paths based on user skills and interests.

## 🌐 Live Demo
https://ai-career-guidance-production.up.railway.app

## ✨ Features
- 🔐 User login & registration
- 🤖 AI-based job suggestions
- 📊 Skill matching percentage
- 🧠 Roadmap generation
- 💡 Missing skills detection

## 🛠 Tech Stack
- Python (Flask)
- Pandas
- Scikit-learn
- HTML, CSS

## 📂 Project Structure
- app.py → Main backend
- templates/ → HTML pages
- requirements.txt → Dependencies
- Procfile → Deployment config

## ⚙️ Installation

```bash
git clone https://github.com/MANSI28405/ai-career-guidance.git
cd ai-career-guidance
pip install -r requirements.txt
python app.py
